Building RyujiNX (Windows only for now):

To get started; you will need Visual Studio 2015 or later. You will also need the .NET Core 2.0 or greater runtime installed.


Step one:

Download the latest version of Visual Studio frome here. Once you have that done, make sure to run the installer, and select the C# Language package(s) and NET Core 2.0.
You may also use NET Core instead of using Visual Studio. Download NET Core, here.


Step two:

After the installation of Visual Studio is done; go ahead and copy the Clone link from GitHub (From here~ via Clone or Download --> Copy HTTPS Link. Or you can download the ZIP tarball.


Step two: Varient one

You can Git Clone the repo by using the GitBash, or you may use the second varient.


Part two: Varient two

Download the ZIP Tarball. Then extract it to a directory of your choice.


Step three: Varient one

Open the vsproj file in Visual Studio, wait for it load the solution; and then switch the Debug build type to release. Then build.
The build directory is "root/bin/release/etc/etc."


Step Three: Varient two

Build the App using a Command prompt in the ROOT directory. You can quickly access it by Holding shift in explorer (in the RyujiNX irectory)
and typing the following command~ "dotnet build -c Release".


The build directory is "root/bin/release/etc/etc."

Part four:

In order to run a NRO or NSO; simply drag the file onto the Executable. The app will launch, and the homebrew/application will being emulation.
Do keep in mind, that emulation is finicky, and will most likely crash at some point.
